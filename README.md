# CodeLeap Test Project

This repository contains a test project developed as part of the recruitment process for a position at **CodeLeap**.  
The goal of this project is to demonstrate technical skills in **React**, **TypeScript**, and **styled-components**, while building a simple CRUD application for posts.

## 📌 Features

- Create, update, and delete posts
- Responsive layout for desktop and mobile
- Integration with the CodeLeap API

## 🛠️ Technologies Used

- **React** (with hooks)
- **TypeScript**
- **Styled-components** for styling
- **Fetch API** for backend communication

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (>= 16)
- npm or yarn

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/codeleap-test.git
cd codeleap-test
npm install
```
